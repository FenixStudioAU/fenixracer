# Fenix AI Games – Fenix Racer

Fenix Racer is part of the **Fenix AI Games** project: a collection of browser-based games created with AI, using simple prompts, plain HTML/JavaScript, and minimal tooling.

The goal is to see how far AI-assisted game development can go without relying on full game engines, complex build systems, or heavy external workflows.

## About the Game

This project is a standalone browser game built using:

* HTML
* CSS
* JavaScript
* Phaser.js and/or Three.js where required
* AI-assisted coding and iteration

The game is designed to run directly in a modern web browser.

## Project Goals

This project is part of an experiment to create feature-complete games using AI with:

* Minimal prompts
* No traditional game engine
* No complicated setup
* Browser-based deployment
* Simple, editable source files
* Fast iteration and testing

Each game may include different mechanics, styles, and experiments depending on the prompt used to create it.

## Features

Depending on the specific game, features may include:

* Title screen and menus
* Keyboard, mouse, or touch controls
* Procedural or hand-built levels
* Enemies and hazards
* Power-ups and collectibles
* Music and sound effects
* Scoring, lives, or progression systems
* Save/load or local storage features
* AI-generated code, assets, and mechanics

## AI Development Notes

This game was created as part of an AI-assisted development experiment.

The process generally involves:

1. Writing a detailed game prompt.
2. Asking an AI model to generate a playable HTML/JavaScript game.
3. Testing the result in the browser.
4. Iterating through fixes, improvements, and new mechanics.
5. Documenting what worked, what failed, and which AI model was used.

The project is intentionally experimental. Some code may be unusual, messy, or imperfect because part of the goal is to test what AI can produce directly.
